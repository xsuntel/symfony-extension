# README

This project is an extention for VSCode to develop a web application using [Symfony Framework](https://symfony.com)

## Project

* Directory Structure

```text
./                                            ← Repository root
app/
├── extension.js                              ← activate/deactivate
├── package.json
├── src/
│   ├── providers/
│   │   ├── completionProvider.js
│   │   ├── hoverProvider.js
│   │   └── definitionProvider.js
│   ├── views/
│   │   ├── servicesTreeProvider.js
│   │   ├── routesTreeProvider.js
│   │   └── parametersTreeProvider.js
│   └── symfony/
│       └── console.js                        ← bin/console
└── test/
```
